「投資智聊 AI」是一個基於人工智慧技術的台股分析平台，提供全方位的投資決策支援工具。透過整合技術面、籌碼面、消息面與基本面的四大分析維度，搭配 AI 智能助理，幫助投資人快速了解股票表現，做出更明智的投資決策。

## ✨ 主要功能

### 🔍 智能股票搜尋
- **快速查詢**：支援股票代碼與公司名稱搜尋
- **台股全覆蓋**：涵蓋台股市場眾多上市上櫃股票
- **搜尋記錄**：自動保存搜尋歷史，方便快速存取

### 📊 股票速覽儀表板
- **即時股價**：顯示即時股價、漲跌幅、成交量等關鍵資訊
- **互動式 K 線圖**：採用 ECharts 提供專業級的股價走勢圖表
- **隔日漲跌預測**：基於 AI 模型分析的隔日上漲機率預測
- **四大面向評分**：一目了然的多維度評估卡片

### 📈 四大維度深度分析儀表

#### 🔧 技術面分析
- 多種技術指標計算與評估（MA、MACD、RSI、KD 等）
- 技術型態辨識

#### 🧮 籌碼面分析  
- 三大法人進出統計與視覺化
- 主力籌碼集中度觀察
- 融資融券變化追蹤
- 大戶持股比例分析

#### 📰 消息面分析
- 相關新聞即時整理
- AI 新聞情緒分數評估
- 關鍵事件標注

#### 💰 基本面分析
- 財務指標深度評估
- 營收獲利趨勢分析
- EPS、ROE、本益比等關鍵數據
- 視覺化呈現個股歷年同月表現

### ☁️ 新聞股雲功能
- **相關性分析**：找出走勢相關性高的個股
- **文字雲視覺化**：以詞雲呈現相關性強度

### 🤖 AI 智聊機器人
- **24/7 投資諮詢**：全天候線上投資問答服務
- **多模型支援**：支援 GPT-4o、GPT-5 等多種 AI 模型
- **會話管理**：完整的對話歷史記錄與管理功能
- **個股深度分析**：針對特定股票提供客製化分析建議
- **投資知識問答**：回答各類投資理財相關問題

### 👤 個人化功能
- **會員系統**：透過 Supabase 提供安全的身份驗證
- **自選股清單**：建立與管理個人化觀察清單

## 🚀 快速開始

### 線上使用
1. **開啟網站**：訪問 [投資智聊 AI](https://investchat.net)
2. **註冊/登入**：建立帳號或使用訪客模式
3. **搜尋股票**：在首頁搜尋框輸入股票代碼或公司名稱
4. **瀏覽分析**：查看股票速覽與四大面向的詳細分析
5. **使用 AI**：點擊右下角「智聊 AI」按鈕開始對話
